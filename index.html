This page dynamically generates a different coffee for each day of the week using JavaScript

Data is stored in a JavaScript object holding coffee data such as a unique image, alt tag, color, text and a description.

Data is loaded to the page using document.querySelector() and document.getElementById()

The coffee also changes when links are clicked for each day of the week.

This is done by accessing the query string using the URLSearchParams() object, and passing in ?day=1 for example.

Thus all 7 days of the week are available using only one web page via the use of JavaScript
